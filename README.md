# ExamIDE-Coding-Platform-for-Placement-Tests
ExamIDE is a web platform for creating and conducting placement tests with both multiple-choice and auto-graded coding problems, built using Flask with PostgreSQL for data management, Flask-SQLAlchemy and Flask-Migrate for ORM and migrations, Flask-WTF and Flask-Login for security, and Jinja2 with HTML, Bootstrap, and JavaScript for the frontend.
